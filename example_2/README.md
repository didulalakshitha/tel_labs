# example_1

Calculating the minimum price plans for given features.

## Specification

openjdk 11.0.15
Gradle 6.8.3

## commands

### To build
`gradle clean build`

### To execute

`gradle run --args="<file_path> <features>"`

Ex:\
`gradle run --args="/home/didula/workspace_dlak/telepathy/example_2/Example1.txt email,voice,admin"`
